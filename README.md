## cop-3701-traffic-accident-risk-analytics
Tenatively, this project aims to create a database that tracks various traffic accident risks analytics including accident severity, weather, and location. The final database should include features like the ability to detect accident hotspots, severity ranking, and spatial-temporal query optimization. The users this database is best suited for those who seek to monitor automotive traffic and use it to prevent or plan around potential accidents. The dataset used for this project was collected by traffic APIs between 2016 and 2023, and compiled by Sobhan Moosavi.

The dataset used for this project is linked below.
